# StorCLI Dashboard

A Streamlit-based dashboard for visualizing and searching RAID controller logs generated by StorCLI.

## Features
- Upload `.txt` or `.zip` log files
- View logs with meaningful titles
- Search logs by keyword
- Automatically maps RAID controller info, VD/PD status

## How to Use
1. Upload your log files
2. Use the search bar to filter by keyword
3. Expand each section to view details

## Requirements
streamlit 
pandas

## Deployment
This app is hosted on [Streamlit Cloud](https://streamlit.io/cloud) and auto-updates with each GitHub push.
